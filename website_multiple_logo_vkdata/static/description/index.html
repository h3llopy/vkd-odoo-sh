Multiple Website Logo

Features of This Moduel

This module allows you to upload seperate Logo and Favicon for multi website.This requires you to setup Odoo multi site first and this module will work with Odoo standard themes only.But we can customize this for your own theme too.So please talk to us!

How to?

Login to Website Admin > Domains > Select your Website and upload the Logo and Favicon
